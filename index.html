About

Uses

Audience

  • Some or significant programming experience
  • Understands core programming concepts

Characteristics

  • Predominantly male users
  • Mid to late 20s
  • United States

Mapping

Features

  • Key-binding frameworks (vim, emacs)
  • Project collaboration (kumite)

Weaknesses

Improvements
